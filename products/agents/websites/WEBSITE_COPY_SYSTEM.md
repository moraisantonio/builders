# WEBSITE_COPY_SYSTEM — Sistema de copy para o agente

> Referência rápida de fórmulas e regras de copy. Para contexto completo: `docs/websites/website-copy-guide.md`.

---

## Fórmula universal de proposta de valor

```
[Nome do negócio] ajuda [perfil do público] a [resultado concreto] sem [dor principal].
```

Esta frase deve guiar: headline, subheadline e CTA principal.

---

## Estrutura de seção (qualquer seção)

```
Headline: [o que / para quem] — máx. 8 palavras
Corpo:    [como + por quê] — máx. 3 parágrafos de 2–3 linhas
CTA:      [verbo + resultado] — máx. 4 palavras
```

---

## Fórmulas de headline por seção

| Seção | Fórmula | Exemplo |
|---|---|---|
| Hero | [Resultado] + [Para quem] | "Projetos que saem do papel." |
| Problema | [Dor como afirmação ou pergunta] | "Portfólio incrível. Site que não convence." |
| Solução | [Como resolve] + [Sem a dor] | "Do conceito ao site, em menos de 3 semanas." |
| Serviços | [O que você ganha] | "O que fazemos pelo seu negócio" |
| Prova | [Credibilidade] | "O que dizem quem trabalhou com a gente" |
| CTA final | [Urgência real ou reforço] | "Pronto para aparecer como merece?" |

---

## Regras invioláveis de copy

1. Resultado antes de processo — sempre
2. CTA: nunca "Saiba mais" ou "Clique aqui"
3. Frase > 2 linhas no mobile → quebrar
4. Sem jargão que o cliente do cliente não usaria
5. Sem clichês: "solução completa", "ecossistema", "potencializar", "alavancar", "robusto"
6. Texto específico > texto genérico — sempre

---

## Adaptação por nicho

### Arquitetura
- Resultado: "projetos que saem do papel", "clientes que entendem o processo"
- Tom: sofisticado, visual, confiante
- Vocabulário: projeto, processo, espaço, entrega, conceito, execução

### Cafeteria
- Resultado: "mais movimento", "cliente que volta", "encontrável no Google"
- Tom: acolhedor, próximo, sensorial, local
- Vocabulário: café, espaço, cardápio, bairro, ambiente, comunidade

---

## Checklist antes de entregar qualquer copy

- [ ] Tem frase com mais de 2 linhas no mobile?
- [ ] Tem clichê ou jargão?
- [ ] O benefício está antes do processo?
- [ ] O CTA diz o que acontece depois do clique?
- [ ] O texto parece escrito para este negócio, não para qualquer um?
- [ ] Tem placeholder?
