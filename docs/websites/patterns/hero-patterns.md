# Hero Patterns — Padrões de seção hero

> Referência de estruturas e abordagens para a primeira seção de qualquer site.
> O hero é a seção mais importante — determina se o visitante fica ou vai embora.

---

## Regras universais do hero

- CTA principal visível sem rolar (above the fold)
- Headline: o que o negócio faz + para quem, em no máximo 8 palavras
- Subheadline: expande o benefício ou elimina a objeção principal — máximo 2 linhas
- Não usar "Bem-vindo" nem o nome da empresa como headline — é espaço desperdiçado
- Um único CTA principal — não três opções concorrentes

---

## Padrão 1 — Headline + Sub + CTA (estrutura mínima)

**Quando usar:** landing pages focadas em conversão, quando o visual é simples e o texto é protagonista.

```
[Headline — resultado + para quem]
[Subheadline — como + eliminando qual dor]
[Botão CTA]
```

**Exemplo:**
```
Projetos que saem do papel.
Criamos a presença digital do seu escritório para atrair o cliente certo
— sem depender de indicação.
[Ver nossos projetos]
```

---

## Padrão 2 — Headline + Sub + CTA + prova (com credibilidade)

**Quando usar:** quando há números reais, clientes conhecidos ou resultados mensuráveis para mostrar.

```
[Headline]
[Subheadline]
[Botão CTA]
[Linha de prova: "X clientes atendidos" / "Y projetos entregues"]
```

**Exemplo (cafeteria):**
```
O café que seu bairro vai procurar no Google.
Site, cardápio e atendimento automático para você focar no que faz de melhor.
[Conhecer o [nome]]
"Mais de 200 avaliações 5 estrelas no Google."
```

---

## Padrão 3 — Split (texto + visual)

**Quando usar:** quando o produto ou serviço é altamente visual (arquitetura, portfólio, comida).

```
[Metade esquerda: Headline + Sub + CTA]
[Metade direita: Imagem ou mockup principal]
```

**Cuidado:** em mobile, a imagem vai para baixo do texto — garantir que o texto sozinho já converte.

---

## Padrão 4 — Full-bleed com overlay (imagem de fundo)

**Quando usar:** quando a identidade visual é forte e tem foto de alta qualidade do espaço.

```
[Foto de fundo full-bleed]
[Overlay escuro: mínimo 30%]
[Logo no topo]
[Headline centralizada em branco]
[Subheadline]
[CTA em botão colorido]
```

**Cuidado:** testar contraste com a ferramenta — overlay insuficiente quebra a acessibilidade.

---

## Padrão 5 — Minimalista (tipografia como protagonista)

**Quando usar:** escritórios de design, arquitetura, marcas premium que confiam na tipografia.

```
[Headline grande, peso leve, espaçamento amplo]
[Linha divisória fina]
[Subheadline pequena, texto secundário]
[CTA como link sublinhado ou botão fantasma]
```

**Cuidado:** requer tipografia de qualidade e foto de fundo ou fundo sólido bem escolhido.

---

## O que nunca fazer no hero

- Slider/carrossel automático com múltiplas headlines — o visitante não lê nenhuma
- Vídeo com autoplay e áudio
- Texto na frente de imagem sem overlay — ilegível
- Headline genérica: "Bem-vindo ao nosso site" / "Soluções para o seu negócio"
- Três CTAs no mesmo nível — cria paralisia de decisão
- Animações de entrada excessivas que atrasam a leitura
