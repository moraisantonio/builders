# Footer Patterns — Padrões de rodapé

> O rodapé é a última chance de converter — e a primeira fonte de informações essenciais para quem sabe o que quer.

---

## Princípio central

O rodapé não é decoração. Quem rola até o final está buscando algo específico: endereço, telefone, horário, Instagram. Não frustrar essa busca.

---

## Padrão 1 — Rodapé simples (para sites de uma página ou landing pages)

```
[Logo]
[Tagline opcional — 1 linha]
[WhatsApp] [Instagram] [E-mail]
[Endereço com link para Maps] (se PME local)
[Horário de funcionamento] (se relevante)
© [Ano] [Nome do negócio]
```

**Quando usar:** sites pequenos, landing pages, negócios com canal único de contato.

---

## Padrão 2 — Rodapé em colunas (para sites multi-página)

```
Coluna 1: Logo + tagline + redes sociais
Coluna 2: Navegação — links principais do site
Coluna 3: Contato — telefone, WhatsApp, e-mail, endereço
[Opcional Coluna 4]: Horário / Localização / Links legais
Linha inferior: © [Ano] [Nome do negócio] · Política de privacidade
```

**Quando usar:** sites com 4+ páginas, escritórios profissionais.

---

## Padrão 3 — Rodapé com CTA (conversão até o fim)

```
[Seção CTA final acima do rodapé: headline + botão]
[Linha separadora]
[Rodapé simples: logo + links + copyright]
```

**Quando usar:** landing pages de alta conversão onde cada visitante que não converteu precisa de mais um empurrão.

---

## Elementos obrigatórios por tipo de negócio

### PME local (cafeteria, escritório)
- [ ] Endereço com link para Google Maps
- [ ] Telefone/WhatsApp clicável
- [ ] Horário de funcionamento
- [ ] Instagram com link correto

### Escritório de arquitetura
- [ ] E-mail profissional clicável
- [ ] WhatsApp direto
- [ ] LinkedIn (se ativo)
- [ ] Instagram com portfólio

### Qualquer site
- [ ] Copyright com ano atual
- [ ] Política de privacidade (obrigatório se coleta qualquer dado — LGPD)
- [ ] Logo com link para home

---

## Elementos opcionais úteis

- Newsletter (apenas se o cliente vai manter uma frequência real de envio)
- Link para o Google Business Profile ("Avalie no Google")
- Selos de certificação ou associação profissional (CAU, AsBEA para arquitetura)

---

## O que nunca fazer no rodapé

- Rodapé vazio com só o copyright — perdeu oportunidade de reter o visitante
- E-mail como imagem — não é clicável
- Telefone sem ser clicável (`tel:`) — no mobile não funciona
- Redes sociais que não têm atividade real — melhor não linkar
- Links para páginas que não existem ou estão em construção

---

*Dica: em mobile, o rodapé com coluna única deve ter os itens mais importantes primeiro (contato, endereço) — não os links de navegação.*
