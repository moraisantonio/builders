# Services Patterns — Padrões de seção de serviços

> Como apresentar o que o negócio oferece de forma clara e que vende.

---

## Princípio central

A seção de serviços não é uma lista de o que você faz. É uma lista de o que o cliente ganha.

**Ruim:** "Design de interiores residencial e comercial com foco em sustentabilidade"
**Bom:** "Projeto completo para seu apartamento — do conceito ao mobiliário, com você envolvido em cada etapa"

---

## Padrão 1 — Cards (mais comum)

**Quando usar:** 3–6 serviços com diferenças claras entre eles.

```
[Headline da seção: "O que fazemos" / "Como podemos ajudar"]

[Grid 2–3 colunas no desktop, 1 no mobile]
  Card:
    - Ícone ou foto (opcional)
    - Nome do serviço (concreto)
    - Descrição: o que é + o que resolve (1–2 linhas)
    - CTA opcional: "Saiba mais" → ou link para página de detalhe
```

**Erro mais comum:** três colunas genéricas com ícone + título + texto idêntico em estrutura. Variar comprimento e destaque dos cards.

---

## Padrão 2 — Lista com expansão (accordion)

**Quando usar:** muitos serviços (7+) ou quando cada serviço precisa de mais explicação.

```
[Lista de serviços colapsáveis]
  → Clique abre descrição completa
  → CTA no interior expandido
```

**Quando não usar:** quando há poucos serviços — accordion com 3 itens parece evasivo.

---

## Padrão 3 — Destaque + lista

**Quando usar:** quando há um serviço principal e outros complementares.

```
[Serviço principal em destaque — mais espaço, foto ou visual]
[Lista dos demais serviços abaixo — mais compactos]
```

Cria hierarquia visual que orienta o visitante para o produto de maior valor.

---

## Padrão 4 — Seção "Como funciona" integrada

**Quando usar:** quando o processo é tão importante quanto o serviço (ex: arquitetura, consultoria).

```
[Serviço descrito]
[Etapas do processo: 1 → 2 → 3]
[CTA: "Começar agora"]
```

Combina o quê com o como — reduz objeções de "mas como vai funcionar?"

---

## Nomear serviços sem jargão

| Jargão técnico | Linguagem do cliente |
|---|---|
| Projeto de interiores residencial | Projeto para sua casa ou apartamento |
| Gestão de presença digital | Cuidar do seu negócio no digital |
| Automação de CRM | Não perder mais lead por falta de resposta rápida |
| SEO on-page | Aparecer no Google quando seu cliente pesquisar |
| Identidade visual | Logo, cores e visual que o cliente reconhece |

---

## O que incluir em cada serviço

- **O que é** — descrição em linguagem simples
- **Para quem** — perfil de quem mais se beneficia
- **O que resolve** — a dor específica que endereça
- **O que inclui** — escopo básico (e o que não inclui, se relevante)
- **CTA** — próximo passo claro
