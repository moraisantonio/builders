# SEO Checklist — Verificação completa antes do launch

> Para detalhes e contexto de cada item, ver `website-seo-guide.md`.

---

## On-page

- [ ] Title tag única por página (50–60 caracteres)
- [ ] Meta description única por página (140–160 caracteres)
- [ ] H1 único por página com keyword principal
- [ ] Keywords aparecem naturalmente no corpo do texto (sem keyword stuffing)
- [ ] Imagens com alt text descritivo e nomes de arquivo significativos
- [ ] Links internos entre páginas relacionadas
- [ ] URLs amigáveis (sem IDs, sem acentos, sem parâmetros desnecessários)

## SEO local

- [ ] Google Business Profile criado/atualizado com todas as informações
- [ ] NAP (nome, endereço, telefone) idêntico no site e no Google Business
- [ ] Schema markup LocalBusiness implementado no código
- [ ] Endereço e telefone no rodapé do site
- [ ] Mapa incorporado ou link para Google Maps na página de contato

## Técnico

- [ ] Sitemap.xml gerado e acessível
- [ ] Robots.txt correto
- [ ] HTTPS ativo
- [ ] Sem erros de crawl (verificar com Google Search Console após indexação)
- [ ] Sem conteúdo duplicado entre páginas
- [ ] Canonical tags nas páginas que precisam

## Velocidade (afeta rankeamento)

- [ ] Imagens comprimidas e em formato moderno
- [ ] LCP (Largest Contentful Paint) < 2,5s
- [ ] CLS (Cumulative Layout Shift) < 0,1

## Pós-launch

- [ ] Site verificado no Google Search Console
- [ ] Sitemap submetido no Search Console
- [ ] Indexação manual solicitada para páginas principais
- [ ] Google Business Profile com link do site atualizado
