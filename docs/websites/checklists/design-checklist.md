# Design Checklist — Antes de apresentar qualquer layout ao cliente

---

## Estratégia visual

- [ ] Modo base definido: escuro, claro ou misto
- [ ] Paleta do cliente definida (mín. cor principal + fundo + texto + CTA)
- [ ] Paleta testada com ferramenta de contraste (mín. 4,5:1 em todos os pares texto/fundo)
- [ ] Tipografia escolhida com escala definida (tamanhos e pesos)
- [ ] Estilo de imagem definido: foto real, banco contextualizado, ou design sem foto

## Layout e composição

- [ ] CTA principal visível sem rolar a página (above the fold)
- [ ] Hierarquia visual clara na home: headline → benefício → prova → ação
- [ ] Espaçamento entre seções consistente (múltiplos de 8px)
- [ ] Grid alinhado — nenhum elemento solto sem razão
- [ ] Versão mobile projetada (375px)
- [ ] Versão mobile não é só "encolheu" — foi pensada para o dedo

## Componentes

- [ ] Botão primário: cor de alto contraste, hover definido, border-radius até 8px
- [ ] Botão secundário: estilo distinto do primário, mas coerente
- [ ] Cards: borda fina (0.5–1px), border-radius 8–12px, sem shadow pesada
- [ ] Formulário: labels explícitas, campos com mínimo de atrito, CTA real no botão

## Proibições verificadas

- [ ] Sem gradientes decorativos
- [ ] Sem sombras pesadas em cards
- [ ] Sem três colunas genéricas com ícones repetidos
- [ ] Sem animações de entrada em todos os elementos
- [ ] Sem mais de três fontes na mesma página
- [ ] Sem carrossel com mais de 3 itens em mobile
- [ ] Sem texto em caixa alta como corpo de texto
- [ ] Sem border-radius > 16px em elementos de UI

## Imagens

- [ ] Fotos reais do negócio estão sendo usadas (ou plano alternativo definido)
- [ ] Nenhuma imagem de stock photo com poses forçadas
- [ ] Imagens com proporção e recorte definidos para cada breakpoint
- [ ] Overlay em fotos com texto: mínimo 30% de escurecimento para legibilidade

## Aprovação do cliente

- [ ] Copy (textos) aprovados antes de finalizar o layout
- [ ] Moodboard ou referências visuais aprovados antes de criar o design completo
- [ ] Paleta e tipografia aprovados como primeiro passo
